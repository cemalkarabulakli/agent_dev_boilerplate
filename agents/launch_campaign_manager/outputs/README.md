# Outputs

This directory stores launch campaign outputs generated by the Launch Campaign Manager agent.

## Output Types

| Type | Filename Pattern | Description |
|------|-----------------|-------------|
| Launch plan | `launch_plan_[offer_name]_[date].md` | Full launch calendar, asset plan, email schedule |
| Asset audit | `asset_audit_[offer_name]_[date].md` | L.O.A.D. gap analysis |
| Email sequence | `emails_[sequence_type]_[offer_name].md` | Drafted email sequences |
| Revenue forecast | `forecast_[offer_name]_[date].md` | Scenario-based revenue model |
| Post-launch debrief | `debrief_[offer_name]_[date].md` | Metrics, learnings, and next launch hypotheses |
| Traffic plan | `traffic_[offer_name]_[date].md` | Organic + paid traffic coordination plan |
