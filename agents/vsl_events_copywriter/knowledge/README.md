# Knowledge — VSL Events Copywriter

This directory contains reference knowledge for the VSL events copywriter agent.

| File | Purpose |
|------|---------|

## Usage

The agent reads these files when constructing each stage to ensure consistency with proven frameworks.
All examples must be adapted to the specific event and avatar provided by the user — never used verbatim.
