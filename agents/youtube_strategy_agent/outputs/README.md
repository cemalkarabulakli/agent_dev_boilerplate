# Outputs

Agent-generated outputs are stored here.

## Output Types

| Type | Filename Convention | Description |
| --- | --- | --- |
| Niche Validation Report | `niche_[name]_[date].md` | Full 5-dimension niche scoring with recommendation |
| Competitor Analysis | `competitor_analysis_[niche]_[date].md` | Full competitor audit with gap map and content calendar |
| Content Calendar | `content_calendar_[channel]_[week].md` | 4-week rolling content plan with keywords |
| Channel Audit | `channel_audit_[channel]_[date].md` | Full channel health assessment with recommendations |
| SEO Review | `seo_review_[video]_[date].md` | Per-video SEO score and optimization recommendations |
| Monetization Plan | `monetization_plan_[channel]_[date].md` | Revenue stack strategy and YPP progress plan |
| Automation Setup | `automation_setup_[channel]_[date].md` | Tool stack configuration and workflow documentation |
