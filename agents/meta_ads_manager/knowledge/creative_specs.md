# Creative Specs and Hook Framework

## Technical Specifications by Placement

### Facebook / Instagram Feed
- Recommended: 1080 × 1350px (4:5) — takes more screen space than square
- Also accepted: 1080 × 1080px (1:1)
- Image: JPG/PNG, max 30MB
- Video: MP4/MOV (H.264), max 4GB (keep under 1GB for reliability)
- Video duration: 1s–240min (performance sweet spot: 15–30 seconds)
- Text overlay: keep under 20% of image area for full distribution

### Stories (Facebook and Instagram)
- Required: 1080 × 1920px (9:16)
- Safe zones: 250px from top, 250–340px from bottom (UI overlay areas — never put key content there)
- Image display: up to 5 seconds
- Video: up to 60 seconds (best: 15 seconds or less)
- Formats: MP4/MOV for video, JPG/PNG for image

### Reels (Instagram and Facebook)
- Required: 1080 × 1920px (9:16 vertical, full-bleed)
- Video: up to 90 seconds (sweet spot: 15–30 seconds)
- Audio: Reels are audio-forward — silent Reels underperform significantly
- Use original audio or sound that fits the content's pacing

### Carousel
- Each card: 1080 × 1080px (1:1) recommended
- 2–10 cards per carousel
- Each card: image max 30MB, video max 4GB
- Each card has its own headline, link, and CTA

## The Hook Framework

### The 3-Second Rule

Scroll speed increased 41% since 2020. The first 3 seconds determine continuation.

**Structure:**
- Second 0–1: Visual of the end result, or a pattern-interrupt
- Second 1–2: One-line payoff / value statement
- Second 2–3: Fast proof (stat, testimonial fragment, social signal)

### Hook Rate Formula
```
Hook Rate = 3-Second Video Views ÷ Impressions × 100
```

| Range | Assessment |
| --- | --- |
| Below 15% | Poor — replace hook |
| 15–20% | Average |
| 20–40% | Good — test scaling |
| 40%+ | Excellent — prioritize in portfolio |

### Hook Type Categories

| Type | Mechanism | Best For |
| --- | --- | --- |
| Benefit-driven | "Get [outcome] in [timeframe]" | Cold traffic |
| Question | Pain question the audience says "yes" to | Problem-aware |
| Statistic | "9/10 customers report..." | Trust-building |
| Storytelling | Narrative opening with relatable character | Brand warmth |
| Before/after visual | Transformation teaser | E-commerce |
| Reverse psychology | "Don't buy this unless..." | Curiosity |
| Secret/tension | "The one thing your X is missing" | Education |
| Surprise | "Forget everything you knew about X" | Disruption |
| Cliffhanger | "Wait until the end to see..." | Completion rate |
| Controversial | "Most people do X wrong. Here's why." | Engagement |
| Community | "Join [N] people who already..." | Social proof |
| UGC/Authentic | Genuine customer voice | Cold prospecting |
| Problem-agitation | Describe the problem viscerally first | Awareness |

### Silent-View Compliance
80%+ of users scroll without sound. Text overlays must carry the complete message without audio. On-screen captions are mandatory for video ads — silent Reels are especially underperforming formats without them.

## Ad Copy Structure

```
[HOOK LINE — 1 sentence: addresses pain or promises outcome]

[BODY — 2–4 sentences: expand hook, social proof, mechanism]

[CTA — explicit action + reason to act]
```

**Character limits for full display (before truncation):**
- Primary text: 125 characters before "See More" on Feed
- Headline: 40 characters
- Description (below headline): 25 characters

**Copy best practices:**
- Mobile-first: short sentences, line breaks, scannable bullets
- Front-load the hook within the first 125 characters
- Avoid prohibited claims: "best," "guaranteed," "instant results," "#1" without proof
- Avoid ALL CAPS in full body (policy violation)
- Emojis increase CTR when used sparingly and relevantly
- Label AI-generated content as required (Meta 2025 policy)

## Dynamic Creative Optimization (DCO) / Advantage+ Creative

DCO automatically tests and combines asset variants to find best-performing combinations per user.

**Asset requirements:**
- Images/Videos: 3–10 total (mix formats)
- Headlines: 3–5 variations
- Primary text: 3–5 variations
- CTA buttons: multiple options

**Advantage+ Creative automatic features (2025):**
- Auto-generated text variations
- Image enhancements (brightness, contrast)
- Music background suggestions
- Aspect ratio adaptation across placements
- Product label overlays

**DCO limitations:**
- Requires $300–500+/day per ad set for statistically meaningful component testing
- Less control over which combination serves to which user
- Not ideal for precision brand consistency campaigns

**DCO best for:** Large product catalogs, high-budget campaigns, e-commerce with many SKUs.

## Creative Testing Priority Order

Test in this order — most impact first:

1. Hook (first 2–3 seconds / opening line) — highest leverage point
2. Format (video vs. static vs. carousel vs. UGC)
3. Offer / value proposition
4. Body copy angle
5. CTA wording
6. Visual treatment (lifestyle vs. product-only vs. infographic)

**Testing structure:**
- Equal budget per test variable (ABO for isolation)
- 7-day minimum test duration
- 50+ conversions or 500+ link clicks per ad for statistical significance
- One variable at a time
- Kill zombie ads: spent > 1× AOV, zero conversions
