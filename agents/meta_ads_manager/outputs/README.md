# Meta Ads Manager Outputs

Generated outputs from the Meta Ads Manager agent. All files in this directory are gitignored by default.

| Output Type | Description |
| --- | --- |
| `campaign_audits/` | Andromeda readiness audits for existing campaigns |
| `creative_briefs/` | P.D.A.-based creative briefs for new ad concepts |
| `campaign_plans/` | Full campaign structure recommendations |
| `performance_reports/` | Weekly performance analysis with fatigue signals |
| `ad_scripts/` | Indirect and direct video ad scripts |
